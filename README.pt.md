# ✨ **Avatax**

Avatax é um **sistema de rastreamento facial em tempo real** que transforma a entrada da sua **webcam em animação de avatar 3D em tempo real**.

![AvataxDemo](https://github.com/KrishBharadwaj5678/Avatax/raw/main/AvataxDemo.png)

## ✨ **Funcionalidades**

| **Funcionalidade**                      | **Descrição**                                                     |
| --------------------------------------- | ----------------------------------------------------------------- |
| ⚡ **Rastreamento Facial em Tempo Real** | Usa MediaPipe FaceMesh para detectar os pontos faciais.           |
| 🎥 **Captura de Movimento pela Webcam** | Funciona apenas com a webcam, sem necessidade de hardware extra.  |
| 🧩 **Avatares Personalizáveis**         | Escolha entre avatares disponíveis no sistema.                    |
| 🎨 **Painel de Controle de Cores**      | Ajuste facilmente iluminação e cores do ambiente.                 |
| 🌀 **Animações Suaves**                 | Interface e transições feitas com Framer Motion (Framer Motion).  |
| 📱 **Interface Responsiva**             | Tailwind CSS garante adaptação perfeita em todos os dispositivos. |
| 🖥️ **Renderização com Three.js + R3F** | Renderização 3D rápida e otimizada com React Three Fiber.         |

---

## ⚙️ **Tecnologias**

| **Tecnologia**        | **Finalidade**             |
| --------------------- | -------------------------- |
| ⚛️ **React.js**       | Construção da interface    |
| 🧩 **TypeScript**     | Tipagem estática           |
| 🎮 **Three.js / R3F** | Renderização 3D            |
| 👁️ **MediaPipe**     | Detecção de pontos faciais |
| 🎞️ **Framer Motion** | Animações da interface     |
