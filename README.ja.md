# ✨ **アヴァタックス**

アヴァタックス は、**リアルタイムの顔トラッキングシステム**で、**Webカメラの入力をリアルタイムで3Dアバターアニメーションに変換**します。

![アヴァタックス](https://github.com/KrishBharadwaj5678/Avatax/raw/main/AvataxDemo.png)

## ✨ **機能**

| **機能**                       | **説明**                                          |
| ---------------------------- | ----------------------------------------------- |
| ⚡ **リアルタイム顔トラッキング**          | MediaPipe FaceMesh を使用して顔のランドマークを追跡します。         |
| 🎥 **Webカメラモーションキャプチャ**      | 追加のハードウェアなしでWebカメラのみで動作します。                     |
| 🧩 **カスタマイズ可能なアバター**         | 内蔵アバターから選択できます。                                 |
| 🎨 **カラーコントロールパネル**          | ライティングや環境カラーを簡単に調整できます。                         |
| 🌀 **スムーズなアニメーション**          | UIとトランジションは Framer Motion（フレーマーモーション）で実装されています。 |
| 📱 **レスポンシブUI**              | Tailwind CSS により、すべての画面サイズで最適に表示されます。           |
| 🖥️ **Three.js + R3Fレンダリング** | React-Three-Fiberを使った高速で最適化された3Dレンダリング。         |

## 🛠️ **インストール方法**

### 1️⃣ リポジトリをクローン

```bash
git clone https://github.com/KrishBharadwaj5678/Avatax.git
```

### 2️⃣ プロジェクトフォルダへ移動

```bash
cd Avatax
```

### 3️⃣ 依存関係をインストール

```bash
npm install
```

### 4️⃣ 開発サーバーを起動

```bash
npm run dev
```
