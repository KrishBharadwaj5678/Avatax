# ✨ **Аватакс**

Аватакс это **система отслеживания лица в реальном времени**, которая преобразует вход с **веб-камеры в 3D-анимацию аватара в реальном времени**.

![Аватакс](https://github.com/KrishBharadwaj5678/Avatax/raw/main/AvataxDemo.png)

## ✨ **Функции**

| **Функция**                                | **Описание**                                                          |
| ------------------------------------------ | --------------------------------------------------------------------- |
| ⚡ **Отслеживание лица в реальном времени** | Использует MediaPipe FaceMesh для определения ключевых точек лица.    |
| 🎥 **Захват движений через веб-камеру**    | Работает только с веб-камерой, без дополнительного оборудования.      |
| 🧩 **Настраиваемые аватары**               | Выбор из встроенных аватаров.                                         |
| 🎨 **Панель управления цветами**           | Лёгкая настройка освещения и цветов окружения.                        |
| 🌀 **Плавные анимации**                    | Интерфейс и переходы реализованы с помощью Framer Motion.             |
| 📱 **Адаптивный интерфейс**                | Tailwind CSS обеспечивает корректное отображение на всех устройствах. |
| 🖥️ **Рендеринг Three.js + R3F**           | Быстрый и оптимизированный 3D-рендеринг с React Three Fiber.          |
